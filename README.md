## Projects

### [Real-Time Event Tracking Pipeline](https://github.com/Moyijo99/realtime-tracker)
A production-style event ingestion and analytics stack simulating how product teams track user behaviour at scale.

- Kafka handles event streaming and decouples ingestion from processing
- Airflow orchestrates hourly ELT from PostgreSQL into ClickHouse
- dbt models clean and aggregate events into analytics-ready marts
- Visualised in Metabase

**Stack:** Python · Apache Kafka · PostgreSQL · Apache Airflow · ClickHouse · dbt · Metabase · Docker

###  [Abuja Weather Intelligence Pipeline](https://github.com/Moyijo99/weather_abuja_pipeline)
An end-to-end batch pipeline that ingests real weather data for Abuja from the Open-Meteo API, loads it into BigQuery, and transforms it with dbt for downstream analysis.

- Modular ingestion script with schema validation dbt
- dbt staging and mart models enforce consistent data contracts
- Airflow (Astronomer) orchestrates daily runs
- Designed to simulate a real-world analytics engineering workflow

**Stack:** Python · Open-Meteo API · Google BigQuery · dbt · Apache Airflow (Astronomer) · Great Expectations
